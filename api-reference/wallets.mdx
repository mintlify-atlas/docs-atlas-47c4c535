## Create Wallet

<RequestExample>
```bash cURL
curl -X POST https://api.karen.dev/api/v1/wallets \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "my-agent-wallet"}'
```
</RequestExample>

<ParamField body="name" type="string" default="external-wallet">
  Optional human-readable name for the wallet
</ParamField>

<ResponseField name="walletId" type="string">
  Unique identifier for the wallet
</ResponseField>

<ResponseField name="name" type="string">
  The wallet name
</ResponseField>

<ResponseField name="address" type="string">
  Solana public key (base58 encoded)
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of wallet creation
</ResponseField>

## Get Balance

<RequestExample>
```bash cURL
curl https://api.karen.dev/api/v1/wallets/WALLET_ID/balance \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>

<ResponseField name="sol" type="number">
  Native SOL balance
</ResponseField>

<ResponseField name="tokens" type="array">
  Array of SPL token balances
  <Expandable title="Token balance object">
    <ResponseField name="mint" type="string">
      Token mint address
    </ResponseField>
    <ResponseField name="symbol" type="string">
      Token symbol (if available)
    </ResponseField>
    <ResponseField name="balance" type="number">
      Raw balance (smallest unit)
    </ResponseField>
    <ResponseField name="decimals" type="number">
      Token decimal places
    </ResponseField>
    <ResponseField name="uiBalance" type="number">
      Human-readable balance
    </ResponseField>
  </Expandable>
</ResponseField>

## Transfer SOL

<RequestExample>
```bash cURL
curl -X POST https://api.karen.dev/api/v1/wallets/WALLET_ID/transfer \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "RECIPIENT_ADDRESS",
    "amount": 0.5
  }'
```
</RequestExample>

<ParamField body="to" type="string" required>
  Recipient's Solana address
</ParamField>

<ParamField body="amount" type="number" required>
  Amount in SOL to transfer
</ParamField>

<ResponseField name="id" type="string">
  Transaction record ID
</ResponseField>

<ResponseField name="signature" type="string">
  Solana transaction signature
</ResponseField>

<ResponseField name="status" type="string">
  Transaction status: pending, confirmed, failed, or blocked
</ResponseField>

## Swap Tokens

<RequestExample>
```bash cURL

  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \

    "outputToken": "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
    "amount": 1.0,

  }'
```

</RequestExample>

<ParamField body="inputToken" type="string" required>
  Input token mint address
</ParamField>

<ParamField body="outputToken" type="string" required>
  Output token mint address
</ParamField>

<ParamField body="amount" type="number" required>
  Input token amount
</ParamField>

<ParamField body="slippageBps" type="number" default="50">
  Slippage tolerance in basis points (50 = 0.5%)
</ParamField>

<ResponseField name="signature" type="string">

</ResponseField>

<ResponseField name="inputAmount" type="number">
  Amount of input tokens used
</ResponseField>

<ResponseField name="outputAmount" type="number">
  Amount of output tokens received
